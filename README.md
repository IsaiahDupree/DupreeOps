# Dupree Ops, LLC - Official Business Website

Official business website for Dupree Ops, LLC. This site serves as a central reference for partners, vendors, and compliance reviewers (Stripe, Twilio, app stores, etc.).

## Features

- Terminal/ASCII art style design
- Business-facing content optimized for verification
- Legal pages (Terms, Privacy, Billing)
- Product listings
- Contact information
- Responsive design

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

Update the following files to customize for your business:

- `pages/index.tsx` - Main page content, products list, contact info
- `pages/terms.tsx` - Terms of Service
- `pages/privacy.tsx` - Privacy Policy
- `pages/billing.tsx` - Billing & Messaging Policy

## Deployment

This site can be deployed to:
- Vercel (recommended for Next.js)
- Netlify
- Any static hosting service (after `npm run build`)
